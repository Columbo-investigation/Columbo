# Columbo
Discover Social Media Accounts Across Popular Korean Platforms with username, email.

## Usage

### Windows
```bash
cd columbo_investigation
```
```bash
python __main__.py "user_id"
```
or
```bash
python __main__.py -e "email"
```

### Linux
```bash
cd columbo_investigation
```
```bash
python __main__.py "user_id"
```
or
```bash
python __main__.py -e "email"
```
